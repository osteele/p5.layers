p5.js Libraries

This project contains libraries for p5.js.

p5.layers

p5.layers simplifies some common use cases for
p5.js Graphics objects. It
makes it easier to use Graphics objects to implement drawing layers, and it
removes the need to add or subtract the g. prefix to draw calls in order to
change them between drawing on the canvas, and drawing on a Graphics.

function draw() {
  beginLayer();
  background(100);
  fill('blue');
  circle(width / 2, height / 2, 100);
  endLayer();
}

p5.vector-arguments

p5.vector-arguments enables the use of
p5.js Vectors with the p5.js
Shape functions.

function draw() {
  let pt = createVector(mouseX, mouseY);
  circle(pt, 100);

  pt.add(100, 200);
  circle(pt, 50);
}
