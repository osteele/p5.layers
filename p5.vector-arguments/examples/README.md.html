Three studies inspired by Huw Messie's Concentric Circles

Demonstrates createDiv().position() and createButton().position() with p5.Vector arguments.

shapes

Demonstrates circle() and rect() with p5.Vector arguments. Move the mouse to draw circles; click to draw a square.

vertices

Demonstrates vertex() with a 2D p5.Vector argument.

webgl

Demonstrates vertex() with a WEBGL canvas and a 3D p5.Vector argument.
